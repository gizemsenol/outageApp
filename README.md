## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Test the Case

In this project, there is a endpoint for testing the case,
After started the project in 3000 port, case can be test with:
if the request success, Response will have 201 status code with created data.
send request to app like mentioned below:

```bash
curl --location --request POST 'http://localhost:3000/api/v1/outage' \
--header 'Content-Type: application/json' \
--header 'Authorization;' \
--data-raw '{
    "outageId": "norwich-pear-tree",
    "date" : "2022-01-01T00:00:00.000Z"
}'
```

!!! x-api-key and base url are read from '.env' file. the '.env' file format must be like below !!!

```bash
X_API_KEY = '<API KEY>'
BASE_URL = '<BASE URL>'
```
